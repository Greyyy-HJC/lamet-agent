# LaMET Agent

为 大动量有效理论（Large Momentum Effective Theory，LaMET）设计的智能体，专门用于处理 LaMET 的数据分析流程，同时也可以作为一个通用的 LQCD 数据分析智能体的原型。

通常而言，一个完整的 LaMET 计算的数据分析流程可以分为以下几个步骤：
- Correlator analysis
- Renormalization
- Fourier transform
- Perturbative matching
- Extrapolation

关联函数分析和外推步骤需要物理学家的专业知识来设计分析策略，而其他步骤相对固定，可以通过软件工具自动化完成。因此，将这两个步骤委托给智能体可以显著提高分析效率。我们期望智能体能够根据数据的特点，选择合适的分析策略，包括数据预处理、拟合函数的选择、拟合区间的选择以及贝叶斯先验参数的选择等。

## Correlator analysis

关联函数分析主要涉及从 LQCD 的关联函数数据中，提取特定量子数的能谱和特定的强子矩阵元。在这一步，我们需要输入关联函数定义，特别是构造关联函数的强子算符定义（夸克味道，量子数），以及需求的物理可观测量（能谱或者矩阵元）。如果需求是能谱，智能体应该可以通过对两点函数数据进行直接的拟合，或者通过求解广义本征值问题（GEVP）来提取能谱。如果需求是矩阵元，智能体应该可以通过适当的构造，从三点函数和两点函数中消去或者拟合额外参数，最终得到矩阵元的数值。同时，智能体需要理解算符中存在的动量对于最终结果的影响。需要注意，这一步需要考虑重采样用于估计最终结果的统计误差。

具体来说，计划首先通过编写关联函数分析的 SKILL.md 直接嵌入现有的智能体，例如 Claude Code 和 CodeX 等，通过调用这些智能体，观察输出结果，探索现有 LLM 智能体的能力边界。如有必要，需要编写额外的工具或者插件来辅助智能体稳定完成分析任务。

以质子 $g_A$ 计算为例，通常智能体能够获取的关联函数包括质子两点函数 $C_\mathrm{2pt}(t)$ 和质子味道改变流三点函数 $C_\mathrm{3pt}(t, \tau)$，其中 $t$ 是源和汇之间的时间距离，$\tau$ 是算符插入的时间位置。通过智能体分析或者使用额外的程序，得到谱分解的结果：
$$C_\mathrm{2pt}(t) = \sum_n |A_n|^2 \exp(-E_n t)$$
$$C_\mathrm{3pt}(t, \tau) = \sum_{n,m} A_n A_m^* g_A \exp(-E_n (t-\tau)) \exp(-E_m \tau)$$
$$R(t, \tau) = \frac{C_\mathrm{3pt}(t, \tau)}{C_\mathrm{2pt}(t)}$$
此时，可以通过：
- 多态联合拟合直接提取出 $g_A$，也可以
- 在 $t \gg 1$ 且 $\tau \gg 1$ 时，基态贡献主导了关联函数，拟合 $R(t, \tau) \approx g_A$ 获得裸的 $g_A$，还可以
- 时间求和方案 $\sum_\tau R(t, \tau) \approx c + t g_A$ 线性拟合出 $g_A$。

这些方案各有各的优缺点，因此需要智能体根据数据的特点，选择合适的分析策略来提取 $g_A$ 的数值。提取了 LQCD 的裸 $g_A$ 之后，还需要进行重整化，才能得到物理的 $g_A^\mathrm{phys} = Z_A g_A$ 数值。即在这一步通过关联函数分析得到的结果，正是下一步重整化的输入。

有时，重整化也可能需要从数据中直接计算得到。例如在 $J/\psi$ 辐射衰变到赝标量介子的过程中，会需要引入矢量流耦合末态光子，此时就需要 $c\bar{c}$ 介子矢量流重整化系数 $Z_V$。这个数值可以直接根据电荷归一化条件，从 $J/\psi$ 或者 $\eta_c$ 的三点关联函数中提取矢量流矩阵元，进而直接得到 $Z_V$ 的数值，而不需要单独进行重整化公式输入。此时，可以直接要求智能体输出矩阵元，之后手动根据电荷归一化条件进行计算。

根据以上分析，SKILL.md 包含以下内容：
- （可选）根据输入的关联函数定义，写出关联函数的谱分解形式。
- 根据输入的关联函数数据，选择合适的分析策略，提取出能谱或者矩阵元的数值。
  - 需要告知智能体主要的分析策略以及相关优缺点，使得智能体能够根据数据的特点进行合理的选择。
  - 需要考虑不同策略可能带来的系统误差。
- 考虑重采样用于估计最终结果的统计误差。

## Extrapolation

外推步骤主要涉及对格点数据进行连续极限，无穷体积和手征极限的外推。在这一步，智能体需要根据数据的特点，选择合适的外推策略，包括选择合适的拟合函数，选择合适的拟合区间，以及选择合适的贝叶斯先验参数等。此时，拟合函数的形式会变得相对重要，并且有一定的选择自由度。智能体需要理解不同的外推策略对于最终结果的影响，并且能够根据数据的特点进行合理的选择。此时，我们的输入是一个基于格点离散化方案的外推项，智能体需要基于这些外推项，添加可能存在的其他项，目的是提高外推的准确性和可靠性。同时，智能体需要考虑重采样用于估计最终结果的统计误差，并且考虑不同合理外推方案之间存在的系统误差。

在一个常规的 LQCD 计算中，我们至少需要考虑如下的外推项：
- 连续极限外推项，与格点间距 $a$ 相关。通常，对于使用了改进方案的格点离散化，外推项的形式可能是 $a^2$，而对于没有使用改进方案的格点离散化，外推项的形式可能是 $a$。
- 无穷体积外推项，与介子质量 $m_\pi$ 和格子边长 $L$ 相关。典型形式为 $\exp(-m_\pi L)$。
- 手征极限外推项，与 pion 介子质量 $m_\pi^2$ 相关。形式与手征微扰论相关。

在以上经典外推项以外，还可能存在一些虽然阶数更高，但是可以显著提高外推准确性的项。例如，在连续极限外推中，可能存在 $a^4$ 的项；在无穷体积外推中，可能存在 $\exp(-\sqrt{2} m_\pi L)$ 的项；在手征极限外推中，可能存在 $m_\pi^4 \log m_\pi^2$ 的项等。智能体需要根据数据的特点，合理地选择是否添加这些项，以及选择合适的拟合区间和贝叶斯先验参数等，以提高外推的准确性和可靠性。

根据以上分析，SKILL.md 包含以下内容：
- 根据输入的外推项，添加可能存在的其他外推项，以提高外推的准确性和可靠性。
- 根据输入的数据特点，选择合适的外推策略，包括选择合适的拟合函数，选择合适的拟合区间，以及选择合适的贝叶斯先验参数等。
- 考虑重采样用于估计最终结果的统计误差，并且考虑不同合理外推方案之间存在的系统误差。
